A Little About Me

Hey, I'm Vishwa. I build interfaces people actually enjoy using.

I'm a frontend engineer with three years of shipping Angular apps for the enterprise world, from low-code platforms and customer-facing loan portals to CRM modules and freshly modernized analytics dashboards. I started out as a web designer turning Figma files into responsive sites, and that designer's eye still shapes how I write code today.

These days I lead the frontend on a senior Angular team, owning the full lifecycle from UI design through development to testing, and I'm just as comfortable reviewing C# backend logic as I am tuning a CSS easing curve. Drop me a line if you want to chat about clean architecture, thoughtful UX, or what's broken on the web that we could fix together.
